4-Bit Arithmetic Logic Unit (ALU)

This project involves the design and implementation of a 4-bit Arithmetic Logic Unit (ALU) capable of performing multiple arithmetic, logic, and control operations. The ALU integrates several ICs to achieve functionality including addition, subtraction, bitwise operations, increment/decrement, complementing, overflow detection, and Loader/Counter operations.

Features & Operations

  • Bitwise operations: AND, OR, XOR
  • Arithmetic operations: Addition, Subtraction
  • Increment & Decrement (for A and B)
  • 1’s and 2’s Complement (for A and B)
  • Loader and Counter functions
  • Overflow detection
  • Comparator (A > B, A < B, A = B)
  • Sequence control using 16×1 multiplexers

Key ICs Used

  • 74157 – Quad 2×1 Multiplexer
  • 7404 – NOT Gate (1’s Complement)
  • 74283 – 4-bit Adder (Addition, Subtraction, 2’s Complement)
  • 7408 – AND Gate
  • 7486 – XOR Gate
  • 7474 – D Flip-Flop (Loader/Counter)
  • 7485 – Comparator
  • 555 Timer – Clock generator
  • 4067 – 16×1 Multiplexer (Operation selector)

Implementation Details

  • Input Selection: 74157 MUX selects between A and B as inputs.
  • Complement Circuits: 1’s complement via 7404; 2’s complement via 74283 with binary 1 addition.
  • Adder/Subtractor: Single 74283 handles both operations via select line.
  • Increment/Decrement: Implemented by adding or subtracting binary 1.
  • Overflow Detection: Using (A3·B3) XOR (Sum3) with 7408 and 7486.
  • Loader/Counter: 7474 D flip-flops store values and count using clock pulses.
  • Clock: 555 timer with passive components for stable pulse generation.
  • Comparator: 7485 compares A and B, LEDs indicate relation.

Conclusion

The implemented 4-bit ALU integrates arithmetic, logical, and control functions into a single modular circuit. Through the use of multiplexers, adders, logic gates, and flip-flops, the design achieves versatility, accuracy, and expandability for digital computing applications.
