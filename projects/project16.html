ESP32 Curtain Mover with Rain Sensor & L298N Motor Driver

Abstract

This project automates curtain movement using an ESP32, rain sensor, and L298N motor driver. Curtains close automatically when rain is detected and can also be controlled manually via push buttons. The system is designed for smart home integration to improve comfort and protect interiors from weather damage.

Objectives

  • Automate curtain closing upon rain detection.
  • Allow manual curtain opening/closing.
  • Drive a DC motor for curtain movement using L298N motor driver.

Hardware Components

  • ESP32 Development Board
  • Rain Sensor Module (YL-83 or equivalent)
  • L298N Motor Driver
  • DC Motor with pulley/gear for curtain movement
  • Manual control push buttons
  • 12V Power Supply

System Design & Functionality

The rain sensor outputs a digital HIGH signal when wet, triggering the ESP32 to close the curtain by rotating the DC motor in one direction. Manual buttons can override the automation to open or close the curtain as desired.

Pin Configuration

  • Rain Sensor OUT → GPIO32
  • Motor IN1 → GPIO26
  • Motor IN2 → GPIO27
  • Manual Open Button → GPIO33
  • Manual Close Button → GPIO25

Key Code Snippet

if (digitalRead(RAIN_SENSOR_PIN) == HIGH) {
    // Rain detected - close curtain
    digitalWrite(MOTOR_IN1, HIGH);
    digitalWrite(MOTOR_IN2, LOW);
} else if (digitalRead(OPEN_BUTTON) == LOW) {
    // Manual open
    digitalWrite(MOTOR_IN1, LOW);
    digitalWrite(MOTOR_IN2, HIGH);
} else if (digitalRead(CLOSE_BUTTON) == LOW) {
    // Manual close
    digitalWrite(MOTOR_IN1, HIGH);
    digitalWrite(MOTOR_IN2, LOW);
} else {
    // Stop motor
    digitalWrite(MOTOR_IN1, LOW);
    digitalWrite(MOTOR_IN2, LOW);
}

Testing & Results

The system detected rainfall instantly, triggering curtain closure in under 2 seconds. Manual control buttons worked reliably, overriding the automated action when needed. Motor movement was smooth with adequate torque for standard curtains.

Conclusion

The ESP32 Curtain Mover with Rain Sensor provides a practical home automation feature, ensuring protection from sudden rain while giving users full manual control. Future improvements include IoT connectivity for remote operation.
