FM Receiver – MATLAB Simulation & Hardware Implementation

This project implements an FM receiver system using both MATLAB/Simulink simulation and hardware circuitry. The system demodulates an FM signal transmitted by a paired transmitter, allowing recovery of the original message signal in both simulated and real environments.

Objectives

  • Simulate FM signal reception and demodulation in MATLAB and Simulink.
  • Analyze signals in time and frequency domains.
  • Build a hardware FM receiver using discrete components.
  • Ensure receiver operates at the same frequency as transmitter.

Theoretical Background

In Frequency Modulation (FM), the carrier frequency varies according to the message signal. The FM receiver reverses this modulation to recover the original audio signal. FM demodulation techniques extract the baseband message while minimizing noise and distortion.

MATLAB Parameters

Sampling Frequency100 kHz
Carrier Frequency20 kHz
Message Frequency1 kHz
Modulation Index5
Signal Duration0.1 s

Simulation Implementation

  • Generate message and FM modulated signals.
  • Analyze FM signal spectrum using FFT.
  • Demodulate using MATLAB’s fmdemod function.
  • Plot and compare message, FM, and demodulated signals.
  • Simulink model includes FM modulator, demodulator, low-pass filter, and measurement blocks.

Hardware Implementation

  • Active Components: 2N304 transistors, voltage regulator.
  • Passive Components: 40 nH inductor, variable capacitor, capacitors (10 μF, 220 μF), resistors.
  • RF Front-End: LC tank circuit for tuning.
  • Speaker output driven by demodulated signal.
  • Powered by a regulated 5 V supply.

Results

  • Simulation: Clean demodulated signal with proper spectral characteristics.
  • Hardware: Stable FM reception and clear audio output.
  • No significant distortion; proper frequency matching achieved with transmitter.

Conclusion

The FM receiver was successfully implemented in both MATLAB simulation and hardware form. The project provided practical insights into analog communication systems, FM demodulation, and real-world RF circuit challenges.
