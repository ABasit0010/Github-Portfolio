Digital Ammeter using ESP32 & ACS712

This project demonstrates the design and implementation of a Digital Ammeter using an ESP32 microcontroller and an ACS712 Hall-effect current sensor. The system measures current in real time and displays it on an OLED screen, with optional IoT capabilities for wireless monitoring.

Objective

To build a compact and accurate current measurement device that can display readings locally on an OLED display and optionally transmit them over Wi-Fi/Bluetooth for remote monitoring.

Hardware Components

  • ESP32 microcontroller
  • ACS712 Current Sensor (5A/20A/30A versions)
  • OLED Display (128×64 SSD1306)
  • 5V power supply
  • Jumper wires and breadboard/PCB
  • Optional shunt resistor for extended range

Calculation Formula

The ACS712 outputs 2.5V at 0A. The current is calculated as:

Current (A) = (Vout − Voffset) / Sensitivity

For ACS712-5A: Sensitivity = 0.185 V/A

Software Design

The ESP32 reads analog voltage from the ACS712, converts it to current, and updates the OLED display. Optionally, readings can be sent over Wi-Fi/Bluetooth.

Testing & Calibration

  • Connect a known load and measure using a commercial ammeter for reference.
  • Adjust offsetVoltage and sensitivity in the code for calibration.
  • Validate readings at different load levels to ensure accuracy.

Conclusion

The digital ammeter successfully measures and displays current in real time with high accuracy. With IoT integration, it can be transformed into a remote monitoring tool for industrial and hobbyist applications.
