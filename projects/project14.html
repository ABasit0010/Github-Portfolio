ESP32 Toxic Gas Detection (MQ7 & MQ135 + Relay Fan Control)

Abstract

This project implements an ESP32-based toxic gas detection system using the MQ7 (Carbon Monoxide) and MQ135 (Air Quality) sensors. The system continuously monitors harmful gases and, when concentrations exceed safe thresholds, automatically activates a fan via a relay module for ventilation. Real-time gas concentration data can also be sent over Wi-Fi for IoT monitoring.

Objectives

  • Detect Carbon Monoxide (CO) and general air pollutants in real time.
  • Activate ventilation fan automatically when gas exceeds set thresholds.
  • Provide both local (serial/OLED) and remote (Wi-Fi) data monitoring.

Hardware Components

  • ESP32 Development Board
  • MQ7 Gas Sensor (CO detection)
  • MQ135 Gas Sensor (NH3, NOx, Benzene, Smoke, CO2)
  • Relay Module (5V/3.3V compatible)
  • 12V DC Fan
  • OLED Display (SSD1306) for local readings
  • Power Supply (5V/12V)

System Design & Functionality

The ESP32 reads analog values from MQ7 and MQ135 sensors through ADC channels. Sensor values are mapped to ppm concentrations after calibration. If any reading exceeds the predefined safety threshold, the ESP32 activates the relay, powering on the exhaust fan.

  • MQ7 Threshold: ~50 ppm CO
  • MQ135 Threshold: ~150 ppm pollutants
  • Data displayed on OLED in ppm.
  • Optional: Send data to cloud (ThingSpeak / MQTT broker) for logging.

Pin Configuration

  • MQ7 Analog Output → GPIO34
  • MQ135 Analog Output → GPIO35
  • Relay IN Pin → GPIO25
  • OLED SDA → GPIO21
  • OLED SCL → GPIO22

Key Code Snippet

if (mq7_ppm > 50 || mq135_ppm > 150) {
    digitalWrite(RELAY_PIN, HIGH); // Fan ON
} else {
    digitalWrite(RELAY_PIN, LOW); // Fan OFF
}

Testing & Results

During testing, the system accurately detected increased CO levels by exposing the MQ7 sensor to lighter gas and measured poor air quality via MQ135 by introducing smoke. The fan activated immediately upon exceeding thresholds and turned off after air cleared.

Conclusion

The ESP32 toxic gas detection system is effective for home, lab, and industrial safety. With simple calibration, it can integrate into IoT monitoring platforms for remote safety alerts, enhancing air quality management.
