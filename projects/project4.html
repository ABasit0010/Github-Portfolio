Real-Time Visualization and Analysis of EEG Signals

This project implements a real-time EEG signal visualization and analysis system using MATLAB. It processes multichannel EEG data, removes power-line interference, extracts standard brainwave frequency bands, and simulates live signal streaming for neurophysiological analysis.

Objectives

  • Load and visualize multichannel EEG data.
  • Remove 50 Hz noise using a notch filter.
  • Extract Delta, Theta, Alpha, Sigma, and Beta bands using bandpass filters.
  • Simulate real-time streaming for the alpha band.

Background

  • EEG Signals: Brain activity measured via scalp electrodes, recorded in microvolts.
  • Noise Sources: Power-line interference, muscle artifacts (EMG), eye blinks (EOG), electrode movement.
  • Brainwave Bands:
    • Delta: 0.5–4 Hz (deep sleep)
    • Theta: 4–7 Hz (meditation)
    • Alpha: 8–12 Hz (relaxation)
    • Sigma: 12–16 Hz (sleep spindles)
    • Beta: 13–30 Hz (active thinking)

Filter Design

  • Notch Filter: 
    designfilt('bandstopiir', ...)
    Removes 50 Hz power-line interference.
  • Bandpass Filters: 
    bandpass(signal, [f_low f_high], fs)
    Isolate specific EEG frequency bands.

Implementation

  • Load and plot raw EEG signals from .mat dataset (5 channels, 500 Hz).
  • Apply IIR notch filter to remove 50 Hz interference.
  • Design FIR/IIR bandpass filters for each EEG band.
  • Simulate real-time streaming using a sliding 1–2 second window for alpha band extraction.
  • Develop MATLAB App Designer GUI for live EEG channel & band selection with PSD plotting.

Results

  • Significant reduction of 50 Hz interference confirmed in filtered data.
  • Distinct waveform morphology observed in each EEG band.
  • Real-time alpha band streaming prototype successfully demonstrated.

Applications

  • Sleep analysis and monitoring.
  • Brain-computer interfaces (BCI).
  • Clinical neurofeedback systems.

Conclusion

This project demonstrates an effective approach for EEG preprocessing, band extraction, and real-time visualization using MATLAB. The developed framework serves as a foundation for advanced EEG-based monitoring and BCI systems.
