Low-Pass Active Filter Design

This project focuses on designing a second-order Butterworth low-pass active filter using the Sallen-Key topology to meet specific attenuation and gain requirements. The filter was simulated in Proteus and MATLAB, and a physical PCB was fabricated using EasyEDA.

Project Requirements

  • Cut-off frequency: 700 Hz
  • Passband gain: 20 dB (±3 dB variation allowed)
  • Attenuation at ≥8.75 kHz: >10 dB
  • Passband frequency range: ≤700 Hz
  • Roll-off: ≥40 dB/decade (second-order filter)

Filter Overview

A low-pass active filter allows low-frequency signals to pass while attenuating higher frequencies. In this design, a second-order Butterworth filter was chosen for its flat frequency response in the passband and sharp roll-off characteristics. The Sallen-Key configuration was used for stability and simplicity.

Component Selection

  • Operational Amplifier: LM741
  • R1 = 6.8 kΩ
  • R2 = 36.3 kΩ (33 kΩ + 3.3 kΩ in series)
  • C1 = 0.01 µF
  • C2 = 0.022 µF
  • Feedback Resistor (Rf) = 90 kΩ
  • Input Resistor (Ri) = 10 kΩ

Design Calculations

The cut-off frequency was calculated using: 

f_c = 1 / (2π√(R1 × R2 × C1 × C2))
Gain requirement: 
Gain = 1 + (Rf / Ri)
The calculated values met the desired 700 Hz cut-off and 20 dB gain.

Transfer Function

H(s) =                      K / (R1R2C1C2) 
       ---------------------------------------------------------------
       s² + s[(R1C1 + R2C2 + R1C1(1–K)) / (R1R2C1C2)] + 1 / (R1R2C1C2)

Simulation

  • Proteus: Verified cut-off frequency and roll-off performance.
  • MATLAB: Generated Bode plot confirming gain stability in passband and desired attenuation at high frequencies.

Performance Evaluation

  • Passband gain: ~20 dB
  • Cut-off frequency: 700 Hz
  • Roll-off rate: >15 dB at 8.75 kHz (meets attenuation requirement)
  • Stable, noise-free output within passband

PCB Fabrication

Designed using EasyEDA with dimensions 80 mm × 70 mm. The PCB was etched using ferric chloride, drilled, and soldered to complete the hardware build.

Conclusion

The designed low-pass active filter successfully achieved the target specifications. With a 700 Hz cut-off and 20 dB gain, the filter is suitable for applications requiring suppression of high-frequency noise while preserving low-frequency signals.
