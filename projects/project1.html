Atmega32 Development Board + Project

This project involved designing a custom Atmega32-based development board PCB and implementing a toll barrier system with sensors, motor control, and embedded software.

Features:

  • Custom PCB design for Atmega32
  • Relay control for toll barrier motor
  • Sensor integration for vehicle detection
  • Embedded C firmware
